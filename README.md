# Team Ruthless CS2 - Веб-сайт команды

Профессиональный веб-сайт для киберспортивной команды CS2 с системой подачи заявок и админ-панелью.

## Функциональность

### Для пользователей:
- Главная страница с информацией о команде
- Статистика команды
- Форма подачи заявки на вступление
- Контактная информация

### Для администраторов:
- Защищенная паролем админ-панель
- Просмотр всех заявок
- Фильтрация заявок по статусу
- Изменение статуса заявок (принято/отклонено)
- Статистика заявок

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- LocalStorage для хранения данных
- Адаптивный дизайн
- Анимации и эффекты

## Установка и развертывание

### Локальная установка:
1. Клонировать репозиторий
2. Открыть `index.html` в браузере

### Развертывание на Netlify:
1. Зайти на [netlify.app](https://netlify.app)
2. Выбрать "Deploy to Netlify"
3. Загрузить папку с файлами или подключить Git репозиторий
4. Сайт будет автоматически развернут

## Настройка пароля администратора

По умолчанию пароль: `admin123`

Для изменения пароля отредактируйте функцию `adminLogin()` в файле `script.js`
